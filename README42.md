# Description
NetPractice is a networking project meant to develop an understanding of network addressing and routing mechanics. The goal of this project is to solve 10 levels of networking puzzles by correctly configuring IP addresses, subnet masks, and routing tables. By troubleshooting broken or incomplete networks, the project gives a good idea of how data packets travel through local subnets and find their way across the internet.

# Instructions
This project does not contain a program made by the student, but comes with an offline url with the purpose of training/testing you the ways of the net.
Simply run ./run.sh or open the index.html file to begin the training/test.
